# MA&Waves

## Concepts
- Trend: 50 SMA slope defines up, down, or flat.
- Cycle: Defined by Stoch %D crossing the 45/55 zone.
- Half-cycle: Cycle high → cycle low or cycle low → cycle high.
- Complete cycle: Cycle high → cycle low → cycle high (or the inverse).
- Wave: Assigned on half-cycle transitions.
- Odd Wave = Impulse Wave (with trend).
- Even Wave = Retrace Wave (against trend).
- Complex retrace wave letters use `a` through `z`.

## Cycle Rules
- Cycle high: Highest high after %D goes above 45 and before it drops below 55.
- Cycle low: Lowest low after %D goes below 55 and before it rises above 45.
- 45–55 is neutral.

## Anchor Rule
- Wave 1 DOWN = first cycle LOW after 50 SMA starts angling DOWN.
- Wave 1 UP = first cycle HIGH after 50 SMA starts angling UP.
- This is not based on higher highs/lows.

## Impulse Waves (Odd)
- Downtrend: wave lows are 1,3,5,7,9...
- Qualify as wave 3,5,7...:
  - Must be a cycle low.
  - Close < prior impulse low.
  - Open <= prior impulse low.
  - 50 SMA stays angling down from wave 1 to that impulse.
- Uptrend: inverted (cycle high, close > prior impulse high, open >= prior impulse high, SMA stays up).

## Retrace Waves (Even)
- No real-body rule.
- Simple retrace: Only one cycle high/low between two impulse waves.
- Complex retrace: Two or more cycle highs/lows between two impulse waves; only the most extreme qualifies for the wave number.
- Downtrend:
  - Wave 2 = highest cycle high between wave 1 and wave 3.
  - Wave 4 = highest cycle high between wave 3 and wave 5.
  - Wave 6 = highest cycle high between wave 5 and wave 7.
- Uptrend:
  - Wave 2 = lowest cycle low between wave 1 and wave 3.
  - Wave 4 = lowest cycle low between wave 3 and wave 5.

## Reset Rule
- If 50 SMA turns flat or reverses, wave count resets to a new wave 1.
- Every wave is a cycle, but not every cycle is a wave.
